=      File description     =
=============================
Book.java - This class represents a book, which has a title, author, year of
 publication and different literary aspects.

Patron.java - his class represents a library patron that has a name and assigns
 values to different literary aspects of books.

Library.java - This class represents a library, which hold a collection of books.
Patrons can register at the library to be able to check out books, if a copy of
the requested book is available.

README

=============================
=          Design           =
=============================
the classes are designed to enable a full representation of a library, such that
a library instance consists of books and patrons, each of those with its own
traits.

=============================
=  Implementation details   =
=============================
to store the data of the books and patrons, ive initialized two arrays. these
are of types Book and Patron. furthermore, an array of borrowed books was created
in order to keep track of the books each patron has borrowed, making sure the
library rules are kept
